# Employee Survey Analysis Dashboard 📊

## 📌 Project Overview
This project features a detailed analysis of **Employee Engagement Surveys** developed using **Power BI**. The dashboard provides a high-level view of organizational health, employee sentiment, and departmental performance to support data-driven HR decisions.

---

### 🚀 Workflow & Technical Steps

* **Data Transformation:** Cleaned and structured survey data to handle various response types (Likert scale) and departmental categories.
* **DAX Measures:** Implemented advanced DAX for key performance indicators:
    * **Total Responses:** 15K+
    * **Completion Rate:** 99% (Exceptional Engagement)
    * **Average Response Score:** 2.98
* **Sentiment Analysis:** Created visualizations to compare "Agree" vs "Disagree" distributions.
* **Role-Based Benchmarking:** Designed a multi-layered bar chart comparing Staff, Supervisor, Manager, and Director average scores by department.

---

### 🔍 Key Insights

* **Organizational Engagement:** Achieving a 99% completion rate provides a highly reliable foundation for strategic decisions.
* **Sentiment Gap:** Analysis reveals a variance in sentiment between leadership (Directors/Managers) and operational staff.
* **Departmental Trends:** Specific sectors like *Human Resources* and *Public Works* were identified as high-priority areas for management follow-up.
* **Core Strengths:** High frequency of "Strongly Agree" responses indicates solid professional alignment in several key departments.
